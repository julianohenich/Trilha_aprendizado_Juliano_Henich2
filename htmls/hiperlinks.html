Hiperlinks

Hiperlink é o mesmo que um link, ligação ou hiperligação dentro de um hipertexto. É uma forma de referenciar algo e com a finalidade de ligar um documento a outro documento ou de ligar uma página a outra.

Marcação de hiperlinks

Utiliza-se a tag <a> com o href dentro que direciona para onde é indicado

Sintaxe:

<a href="caminho"> texto </a>

Hiperlinks externos

Usado para ir para outra pagina em qualquer parte da internet

Sintaxe:

<a href="URL"> texto </a>

Hiperlinks internos

Usado para ir para algum lugar na mesma pagina em que está

Sintaxe:

<a href="#nomereferencia"> texto </a>

Depois utiliza o comando onde será redicionado

Sintaxe:

<a name="#nomereferencia"> </a>

No lugar do a name também pode se utilizar id:

Hiperlinks locais

Usado para ir para outra pagina que pertence ao mesmo site

Sintaxe:

<a href="arquivo.html"> texto </a>

Hiperlinks em imagens

Usado em alguma imagem para ir para outra pagina ou abrir a mesma imagem ou outra imagem

Sintaxe:

<a href="local"> <img src="imagem.png"> </a>

Hiperlinks para downloads

Usado para baixar arquivos por meio de um hiperlink

Sintaxe:

<a href="nomearquivo.extensao" download="nomearquivo.extensao" type="tipo"> Conteudo </a>

Hiperlinks para endereço eletrônico

Link que direciona para endereço eletrônico, como email

Sintaxe:

<a href="mailto:email@email.com"> Email </a>

Como abrir o link em outra pagina

Utiliza-se o comando target="_blank" dentro da tag <a>

Sintaxe:

<a href="caminho" target="_blank"> texto </a>
