Branches(braços)

São ramificações de um projeto, tem como objetivo desenvolvimento paralelo do projeto.

Braço principal e secundario

As alterações podem ser usadas no braço principal ou secundario.

O principal já altera o projeto em funcionamento enquanto no secundario trabalha de modo separado depois junta os arquivos

Cria uma branch secundaria para trabalhar separadamente, sem afetar o projeto principal depois faz um merge, que serve para colocar as alteraçoes da branch secundaria na branch principal. Pode ser criadas varias branchs.

Hospedagem

Pode-se hospedar um site no GitHub Pages de forma gratuita.

O repositorio deve estar público para hospedar o site

Depois de criado o repositorio vai em settings, depois em pages, escolhe a opção main e save. Será disponibilizado um link que armazena o site
