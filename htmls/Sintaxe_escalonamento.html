Software de escalonamneto

Utilizado para controlar as versões de um código ou documento.
Sendo usado por apenas um desenvolvedor ou uma equipe

Utiliza um sistema de versãoes, criando as novas versões do projeto.
Mantém salvo na nuvem e localmente

Tipos

Centralizado
Repositório central em um servidor
Distribuido
Repositório local com atualização no repositório central

Programas

Programa de controle

Link Git

Plataforma de hospedagem

Link GitHub

Comandos

commit
Uma alteração se torna permanente no local
push
Envia alterações para o repositório remoto
fetch
Verifica sincronia entre os repositórios local e remoto
pull
Faz a sincronia dos repositórios local e remoto
